Personal Access Token
GitHub → Settings → Developer Settings → Personal Access Tokens → Tokens (classic) → scope: repo
Como criar o token
  1. Acesse github.com → Settings
  2. Role até Developer Settings (no final da sidebar)
  3. Clique em Personal Access Tokens → Tokens (classic)
  4. Clique em Generate new token (classic)
  5. Nome: fmc-am-6f-data
  6. Expiration: No expiration
  7. Scope: marque apenas repo
  8. Clique em Generate token e copie o token
  9. Cole no campo acima e clique em SALVAR TOKEN
⚠ O token fica salvo apenas no localStorage do seu browser. Nunca vai para o código nem para o repositório.
